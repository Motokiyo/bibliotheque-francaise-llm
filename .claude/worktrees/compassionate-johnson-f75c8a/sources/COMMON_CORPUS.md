# Common Corpus (Pleias) — Dataset de référence

## Qu'est-ce que c'est ?

Projet Pleias (France) : plus grand dataset open pour LLM entraîné uniquement sur données ouvertes.

## Chiffres clés

- **Taille totale** : 2 000+ milliards de tokens (500 milliards de mots)
- **Part française** : 110 milliards de mots
- **Format** : Parquet (Apache Parquet)
- **Licence** : Domaine public (revendiqué)
- **Support** : Mozilla Builders, HuggingFace, Eleuther AI

## Sous-corpus

| Sous-corpus | Description |
|-------------|-------------|
| OpenCulture | Livres + presse du domaine public |
| OpenGovernment | Documents gouvernementaux |
| OpenScience | Publications scientifiques |
| OpenWeb | Contenu web ouvert |
| OpenSource | Code source |

## French-PD-Books (complément)

- 289 000 livres depuis Gallica (BnF)
- 16,4 milliards de mots
- OCR non nettoyé (qualité ~60-85%)
- Utiliser OCRonos-Vintage pour correction
